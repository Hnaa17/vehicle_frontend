## Getting Started
### Installation
1. Clone this repo (vehicle_frontend)
   ```sh
   git clone https://github.com/Hnaa17/vehicle_frontend.git
   ```
2. Go To Folder Repo
    ```sh
    cd vehicle_frontend
    ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Start website
    ```sh
    npm start
    ```
